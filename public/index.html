Available Endpoints

Available Endpoints.

GET /api View the API summary. GET /api/books Return the full ShelfSpace collection. GET /api/books/:id Return a single book using its numeric ID.
POST /api/books Create a new book, with either an image path or an uploaded cover file.
DELETE /api/books/:id Remove a book from the collection by its numeric ID.
